# InboxAI

AI that manages your inbox

InboxAI is your intelligent inbox assistant that organizes, prioritizes, and helps you manage all your emails, documents, and communications effortlessly. The system searches across multiple sources like emails, documents, and structured datasets to answer user questions.

---

## Features

- AI chat interface
- multi-source document search
- email dataset integration
- document uploads
- vector embeddings
- agent-based automation

---
